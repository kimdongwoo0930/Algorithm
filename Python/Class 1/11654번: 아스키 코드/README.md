# 11654번: 아스키 코드

## 문제 설명


<p>알파벳 소문자, 대문자, 숫자 0-9중 하나가 주어졌을 때, 주어진 글자의 아스키 코드값을 출력하는 프로그램을 작성하시오.</p>



## 입력 형식


<p>알파벳 소문자, 대문자, 숫자 0-9 중 하나가 첫째 줄에 주어진다.</p>



## 출력 형식


<p>입력으로 주어진 글자의 아스키 코드 값을 출력한다.</p>
