# 11050번: 이항 계수 1

## 문제 설명


<p>자연수 \(N\)과 정수 \(K\)가 주어졌을 때 이항 계수 \(\binom{N}{K}\)를&nbsp;구하는 프로그램을 작성하시오.</p>



## 입력 형식


<p>첫째 줄에 \(N\)과 \(K\)가 주어진다. (1 ≤ \(N\) ≤ 10, 0 ≤ \(K\) ≤ \(N\))</p>



## 출력 형식


<p>&nbsp;\(\binom{N}{K}\)를&nbsp;출력한다.</p>



## 예제

### 예제 입력 1

```
5 2

```

### 예제 출력 1

```
10

```
