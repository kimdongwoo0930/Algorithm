# 1978번: 소수 찾기

## 문제 설명


<p>주어진 수 N개&nbsp;중에서&nbsp;소수가 몇 개인지 찾아서 출력하는 프로그램을 작성하시오.</p>



## 입력 형식


<p>첫 줄에 수의 개수 N이 주어진다. N은 100이하이다. 다음으로 N개의 수가 주어지는데 수는 1,000 이하의 자연수이다.</p>



## 출력 형식


<p>주어진 수들 중 소수의 개수를 출력한다.</p>



## 예제

### 예제 입력 1

```
4
1 3 5 7

```

### 예제 출력 1

```
3

```
